# Expense Leak Detection System

This project focuses on analyzing student transaction-level expense data to identify hidden expense leakages caused by small, recurring, and inefficient spending patterns. While individual expenses may seem insignificant, their accumulation over time can result in substantial financial loss.

## 📌 Project Objective
The goal of this system is to help students understand their spending behaviour and detect unnecessary expenses using data analysis, machine learning, and OCR technologies.

## 🚀 Features
- Upload bill / receipt images
- Extract transaction details using OCR
- Automatically categorize expenses using ML model
- Visualize spending insights
- Detect hidden expense leak patterns

## 🧠 Technologies Used
- React Native (Expo)
- Flask (Backend API)
- Python
- Machine Learning (Scikit-learn / Pandas / NumPy)
- OCR (Tesseract)

## 📱 Mobile App
The mobile application allows users to:
- Capture or upload bill images
- View categorized expenses
- Analyse spending trends
